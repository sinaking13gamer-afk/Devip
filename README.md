## قابلیت‌های کامل به تفکیک

---

### احراز هویت

**صفحه لاگین (یکی برای همه):**
- ورود با ایمیل + رمز عبور
- ورود با کد OTP ایمیل (قالب HTML گرافیکی)
- ورود با کد OTP پیامک (SMS.ir)
- ورود با گوگل (OAuth 2.0)
- ورود با گیتهاب (OAuth 2.0)
- Remember Me با Secure Cookie
- Rate Limiting: حداکثر 5 تلاش، بعد 15 دقیقه قفل
- پیام مجزا برای حساب مسدود/تعلیق
- بعد از لاگین: redirect هوشمند بر اساس نقش

**ثبت‌نام:**
- فیلدها: نام، نام خانوادگی، شماره تلفن، ایمیل، رمز، تکرار رمز (همه الزامی)
- نام کاربری = ایمیل
- Validation کامل سمت سرور
- پیامک خوش‌آمدگویی شخصی (SMS.ir)
- ایمیل خوش‌آمدگویی با قالب HTML گرافیکی
- کیف پول پیش‌فرض غیرفعال

**شناسه کاربران:**
- عددی خالص با zero-padding: `000001`, `000002`, ...
- خودکار افزایشی

**توکن‌ها:**
- فرمت: `dvip-XXXXXXXXXX:YYYYYYYYYYYYYYYYYYYYY`
- `XXXXXXXXXX` = شناسه 10 رقمی عددی کاربر
- `YYYYYYYYYYYYYYYYYYYYY` = رشته تصادفی ترکیبی (عدد+حرف)
- کل: 35 کاراکتر
- نام توکن یکتا per کاربر (نمی‌شه دوبار یه اسم داد)

---

### سیستم تیکت (کامل)

**ایجاد تیکت توسط کاربر:**
- دسته‌بندی: مالی / فنی / عمومی / پشتیبانی / سایر
- اولویت: کم / متوسط / زیاد / فوری
- موضوع (عنوان)
- پیام اولیه
- آپلود فایل/تصویر ضمیمه
- بعد از ارسال: redirect به صفحه چت

**صفحه چت تیکت:**
- نمایش پیام‌ها به شکل حباب چت
- ارسال پیام جدید
- آپلود تصویر با پیام
- نمایش زمان هر پیام
- نمایش وضعیت تیکت در بالا
- اگه تیکت بسته شده باشه و کاربر پیام بده: تیکت خودکار باز میشه

**وضعیت‌های تیکت:**
- باز / در حال بررسی / پاسخ داده شده / بسته

**پنل ادمین - مدیریت تیکت:**
- لیست با فیلتر وضعیت/دسته/اولویت
- صفحه پاسخ: چت + تغییر وضعیت + تغییر اولویت در یک صفحه
- ارسال تیکت از طرف ادمین به کاربر
- نمایش اطلاعات کاربر کنار چت

---

### کیف پول

**کاربر:**
- پیش‌فرض غیرفعال، فعال‌سازی با تیکت
- شارژ از طریق ZarinPal
- درخواست برداشت: شماره شبا (IR ثابت) + شماره کارت
- تاریخچه تراکنش‌ها
- مشاهده رسید برداشت (عکس رسید از ادمین)

**ادمین/مدیریت:**
- شارژ/کسر دستی
- فعال/غیرفعال کردن کیف پول هر کاربر
- مدیریت درخواست برداشت (تایید/رد/ویرایش)
- ارسال رسید عکس به کاربر
- ارسال توضیح در قالب تیکت

---

### پنل کاربر - کامل

**داشبورد:**
- خوش‌آمدگویی شخصی
- آمار: تعداد توکن‌ها، موجودی کیف پول، تیکت‌های باز، سرویس‌های خریداری شده
- نمودار فعالیت (خطی + دایره‌ای)
- اعلان‌های اخیر
- دسترسی سریع به بخش‌ها

**پروفایل:**
- ویرایش نام، نام خانوادگی، شماره تلفن
- آپلود/تغییر آواتار
- نمایش شناسه با zero-padding
- نمایش تاریخ عضویت
- وضعیت پریمیوم و تاریخ انقضا

**تنظیمات حساب:**
- تغییر رمز: رمز فعلی + رمز جدید + تکرار (دکمه نمایش هر 3 فیلد)
- تنظیمات اعلان

**نشست‌ها:**
- لیست همه ورودها
- نام دستگاه، IP، مرورگر، سیستم‌عامل
- تاریخ ورود (فارسی)
- وضعیت: آنلاین / آخرین بازدید
- دکمه خروج از نشست خاص

**سرویس‌ها:**
- نمایش فروشگاهی (عکس + نام + قیمت + توضیحات)
- صفحه جزئیات سرویس
- خرید با کیف پول یا ZarinPal
- اعمال کوپن تخفیف
- مشاهده مستندات سرویس

---

### پنل ادمین - کامل

**داشبورد:**
- آمار کلی: کاربران، تیکت‌های باز، درآمد، تراکنش‌ها
- نمودار خطی کاربران جدید (روزانه/هفتگی/ماهانه)
- نمودار دایره‌ای وضعیت تیکت‌ها
- کاربران آنلاین real-time
- آخرین فعالیت‌ها

**مدیریت کاربر (جزئیات کامل):**
- همه اطلاعات شخصی قابل ویرایش
- تعلیق/رفع تعلیق (session kill فوری)
- حذف حساب
- پریمیوم مدت‌دار (تاریخ+ساعت شروع و پایان فارسی) یا دائمی
- ادمین کردن
- مدیریت توکن‌هایش (مشاهده/حذف)
- کیف پول: شارژ/کسر/فعال/غیرفعال
- ارسال تیکت به کاربر
- ارسال ایمیل اختصاصی
- ارسال پیامک اختصاصی
- مشاهده نشست‌هایش
- مشاهده تراکنش‌هایش
- ری‌ست رمز

**مستندات:**
- آپلود فایل‌های .md
- نمایش گیتهاب‌وار (Markdown render)
- Syntax highlighting برای کدها (js, py, php, html, ...)
- مرتب‌سازی و دسته‌بندی

**مدیریت سرویس‌ها:**
- عکس پیش‌نمایش
- نام، توضیحات، قیمت، دسته‌بندی
- وضعیت: فعال/غیرفعال
- لینک به مستندات
- آمار خرید

**لاگ‌ها:**
- نمایش ترمینال‌وار در کادر تاریک
- فیلتر بر اساس نوع (error/info/warning)
- Refresh خودکار
- جستجو در لاگ‌ها

---

### پنل مدیریت (Super Admin) - کامل

**داشبورد آنالیتیک پیشرفته:**
- کاربران آنلاین real-time
- نمودارهای پیشرفته (خطی + دایره‌ای + میله‌ای)
- درآمد روزانه/ماهانه
- آمار تیکت‌ها، سرویس‌ها، تراکنش‌ها
- وضعیت سرور (CPU/RAM/Disk اگه ممکن بود)

**مدیریت ادمین‌ها (RBAC کامل):**
- اضافه/حذف/ویرایش ادمین
- دسترسی per بخش (checkbox):
  - مدیریت کاربران
  - تیکت‌ها
  - سرویس‌ها
  - مستندات
  - تراکنش‌ها
  - برداشت‌ها
  - لاگ‌ها
  - اعلان‌ها
  - پیامک/ایمیل
  - گزارش‌ها
  - کیف پول
  - داشبورد
- دادن همه دسترسی‌های Super Admin به یه ادمین
- تعلیق/رفع تعلیق ادمین

**مدیریت پلن‌ها:**
- Bronze / Silver / Gold
- تعریف محدودیت‌ها per پلن:
  - تعداد توکن مجاز
  - سقف تراکنش
  - دسترسی به سرویس‌های خاص
- فعال/غیرفعال هر پلن

**مدیریت کوپن‌ها:**
- ساخت کوپن با کد دلخواه
- نوع: درصدی یا مقداری
- تاریخ انقضا
- محدودیت استفاده (تعداد کل / per کاربر)
- فعال/غیرفعال
- آمار استفاده

**ارسال همگانی:**
- پیامک همگانی: همه / بر اساس پلن / انتخابی
- ایمیل همگانی: با ادیتور HTML
- پیش‌نمایش قبل از ارسال
- تاریخچه ارسال‌ها

**مدیریت قابلیت‌ها (Feature Flags):**
- فعال/غیرفعال کردن هر قابلیت سیستم
- کنترل نمودارهای داشبورد
- کنترل دسترسی‌های ادمین‌ها به صورت کلی

**IP Manager:**
- Whitelist: فقط این IPها دسترسی دارن
- Blacklist: این IPها مسدود
- گزارش IPهای مشکوک

**تنظیمات سایت:**
- نام سایت، لوگو، فاویکون، توضیحات
- حالت تعمیرات (با پیام سفارشی)
- حالت توسعه
- حالت غیرفعال کامل
- تنظیمات SMTP ایمیل (test connection)
- تنظیمات SMS.ir (API Key + test)
- تنظیمات ZarinPal (Merchant Code)
- توکن‌های پیش‌فرض
- اطلاعات کامل دیتابیس در کادر
- تغییر رمز Super Admin با دکمه نمایش

**Audit Log:**
- هر کار ادمین/Super Admin ثبت
- نمایش ترمینال‌وار
- فیلتر بر اساس کاربر/نوع عملیات/تاریخ

**پشتیبان‌گیری:**
- دانلود backup کامل دیتابیس
- تاریخچه backup‌ها
